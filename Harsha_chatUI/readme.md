# ChatAI – Modern Chat UI

A modern, responsive chat user interface inspired by ChatGPT and Claude, built using HTML, CSS, JavaScript, jQuery, and Bootstrap 5.

---

## Features

### Core Features

* Interactive chat interface with user and AI messages
* Welcome screen with clickable suggestion cards
* Dynamic chat history with multiple conversations
* Auto-saving chat system with no data loss
* Sidebar with chat switching functionality

### UI & UX

* Clean, modern design inspired by ChatGPT
* Smooth animations and transitions
* Rounded message bubbles with avatars
* Typing indicator with animation
* Auto-resizing input textarea

### Functionality

* Send messages using button or Enter key
* Shift + Enter for new line
* Auto-scroll to latest message
* Input validation (no empty messages)
* Timestamp for each message

### Bonus Features

* Dark mode toggle with smooth transition
* Message formatting (code blocks, bold, italic)
* Profile dropdown menu
* Responsive design for mobile and desktop

---

## Technologies Used

* HTML5 – Structure and semantic layout
* CSS3 – Styling, animations, variables
* JavaScript (ES6) – Functionality
* jQuery – DOM manipulation
* Bootstrap 5 – Responsive design
* Font Awesome – Icons

4. The chat UI will open in your browser
